Range Property Check Algorithms

There are three property check algorithms in range-v3 and standarc c++

  • all_of
  • none_of
  • any_of

All these algorithims take an unary predicate and apply on range. all_of checks if the unary predicate is true for all elements, none_of checks the opposite and any_of checks at-least one is true.

Boost range doesn't have any equivalent functions like this.

#include <range/v3/core.hpp>
#include <range/v3/algorithm/all_of.hpp>
#include <range/v3/algorithm/any_of.hpp>
#include <range/v3/algorithm/none_of.hpp>

#include <iostream>
#include <vector>
#include <algorithm>

auto printFunc = [](auto x) { std::cout << x << ',';};

int main()
{
    std::vector<int> v{2, 4, 6};

    std::cout << "v: ";
    std::for_each(v.begin(), v.end(), printFunc);
    std::cout << '\n';

    std::cout << std::boolalpha;

    std::cout << "std::all_of evens: " << std::all_of(v.begin(), v.end(), [](const int& e) { return e%2 == 0;}) << '\n';
    std::cout << "std::any_of divisible by 4: " << std::any_of(v.begin(), v.end(), [](const int& e) { return e%4 == 0;}) << '\n';
    std::cout << "std::none_of odd: " << std::none_of(v.begin(), v.end(), [](const int& e) { return e%2 == 1;}) << '\n';

    std::cout << "ranges::all_of evens: " << ranges::all_of(v, [](const int& e) { return e%2 == 0;}) << '\n';
    std::cout << "ranges::any_of divisible by 4: " << ranges::any_of(v, [](const int& e) { return e%4 == 0;}) << '\n';
    std::cout << "ranges::none_of odd: " << ranges::none_of(v, [](const int& e) { return e%2 == 1;}) << '\n';

    return 0;
}

Output

v: 2,4,6,
std::all_of evens: true
std::any_of divisible by 4: true
std::none_of odd: true
ranges::all_of evens: true
ranges::any_of divisible by 4: true
ranges::none_of odd: true
